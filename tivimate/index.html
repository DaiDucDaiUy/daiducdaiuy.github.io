📺 Hướng dẫn cài đặt

TiviMate 5.1.16

Thực hiện đúng 4 bước bên dưới là xong 🎉

Required:
📱 Android TV 9.0+
🖥️ TV Devices Only
📂 ES File Explorer
1
Tải file về TV
Mở trình duyệt Cốc Cốc TV → Tải file .zip sau: 🔗 https://tr.ee/tivimate
Mở ES File Explorer → Truy cập thư mục Download:
📁 Internal Storage / Download
Giải nén để lấy file .apk và file Backup Host.
2
Cài đặt TiviMate
Chọn file TiviMate 5.1.6 __BannerM0d_Spydog.apk và cài đặt.
⚠️ Không mở ứng dụng sau khi cài xong.
3
Khôi phục Backup
Chọn file Backup Host → khi cửa sổ hiện ra, chọn lần lượt:
✅ TiviMate ✅ This time only
💡 Nếu TV yêu cầu quyền Display over other apps cho TiviMate
→ Truy cập vào Settings cấp quyền rồi quay lại.
4
Cài đặt Playlist
Vào Settings → Playlists — bạn sẽ thấy nhiều danh sách có sẵn, tên đều là xtream-ie.com.
Chọn một danh sách bất kỳ → thay URL bằng link IPTV của bạn.
🚀

Mở TiviMate và tận hưởng thôi!
